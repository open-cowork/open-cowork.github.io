# Open-Cowork Landing Page

The official static landing page for [Open-Cowork](https://github.com/open-cowork/open-cowork).

## Features
- Artistic, high-contrast design
- Bold condensed typography
- Responsive layout
- Pure static implementation

## Tech Stack
- HTML5
- Tailwind CSS (via CDN)
- Google Fonts (Archivo Narrow, Inter)
